# Telco-Churn-Prediction 

Project Overview:
This project develops a machine learning model to predict customer churn in a telecom company. Predicting churn allows businesses to identify customers at risk of leaving and target retention strategies more effectively.

Business Problem:
Telecom providers lose revenue when customers cancel subscriptions. The objective is to predict which customers are likely to churn so retention strategies can be targeted effectively.

Dataset: 
Dataset is "Telco Customer Churn" obtained from Kaggle. Includes: 
7043 customers 
21 attributes 
Target variable: Churn
